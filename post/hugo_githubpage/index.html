Ubuntu下利用Hugo+Github Page搭建个人博客

Github作为最流行的版本控制平台，在码农圈、科研界甚为流行。Github不仅是一个代码存放空间，更提供了开源书籍创作平台Gitbook和个人博客搭建平台Github Page. 本博客便是基于最流行的Hugo架构并借助Github Page发布的。

0.基本介绍

  • 基于Ubuntu18.04 LTS系统
  • Hugo 0.47
  • Hugo Theme: Even
  • 评论系统：valine

1.Github注册与创建Repositories

这部分不多说，Google即可，将两个关键点

  • Github Education Packages，可以了解以下这个东西，直接google
  • Repositories的名字一定要是username.github.io,其中username必须是你的github的用户名，比如我的就是sunningzh

如何创建Repositories参见 https://lucumt.info/post/create-website-with-hugo/

2.安装Hugo

Windows, mac os, Linux都有现成的安装包，傻瓜操作，链接如下：

http://www.gohugo.org/doc/overview/installing/

3.创建你的博客站点

你在电脑创建一个文件夹，hugoblog

然后

$ cd hugoblog
$ hugo new site sunning #最关键的一步，会在hugoblog文件夹下新建一个叫做sunning的文件夹
$ cd sunning
$ ls

4.选择主题并安装

确定当前所在文件夹是sunning，这里我使用的主题是even

git clone https://github.com/olOwOlo/hugo-theme-even themes/even

5.主题配置

将themes/even/exampleSite/config.toml复制到sunning目录下替换原有的config.toml

修改config.toml即可

添加页面，比如我要添加一个关于页面和留言页面

cd sunning
hugo new about.md
vim about.md
#修改about.md内容如下
title: "关于"
menu: "main"
weight: 50

6.本地预览

此时，虽然没有上传到github上，但是可以在浏览器里预览一下，直接输入

hugo server

然后在浏览器里访问 http://localhost:1313/即可

ctrl+c退出预览

7.同步到Github

cd sunning
hugo
cd public
git init
git remote add origin https://github.com/yourname/yourname.github.io.git
git add -A
git commit -m "First submit"
git push -u origin master

然后打开浏览器，输入https://yourname.github.io/ 即可

8.写文章

cd sunning
hugo new post/***.md

只支持markdow，推荐使用typora编辑markdown

9.文章自动分类

Hugo最强大的地方在于，你只管提交文章，它会自动帮你把文章分类。这是如何做到的？每个md文件前面都有一个叫做Font-Matter的东西，比如这篇文章是：

title: "Ubuntu下利用Hugo+Github Page搭建个人博客"
date: 2018-08-26T16:49:30-07:00
draft: false#是否作为草稿，暂不发布
keywords: []
description: ""
tags: [“Hugo”,"Github","Log"]#标签
categories: [“Hugo”]#分类
author: "Sunning"#作者
comment: true#是否开启评论区
......

这些信息你是可以改的，如何hugo根据你填的信息进行分类之类的

10.添加评论系统

有点小复杂，国内比较好的评论系统是valine, 不用登录就能评论，有位大佬已经把过程讲的很清楚了，链接如下：https://www.smslit.top/2018/07/08/hugo-valine/

11.小结

Github入门门槛很高，但一旦学会，大大提高效率。初学最好有人指导，像我自己摸索就走了很多弯路，有问题欢迎联系我。
