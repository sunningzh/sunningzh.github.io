什么叫网络爬虫？

网络爬虫（又被称为网页蜘蛛、网络机器人），是一种按照一定的规则，自动抓取万维网信息的程序或者脚本。《Python网络爬虫开发与项目实战》范传辉 著

百度搜索引擎就属于网络爬虫中的一种。通俗地说，网络爬虫就是按照你的意愿，在广阔的万维网上自动下载并整理你所需要的资源。这样听上去是不是既有趣又有用？培育爬虫是一个艰难的过程，万维网时刻在变化，网络爬虫要想生存下来，必须时刻进化！

0.动机

这就是一份读书笔记。

大约是在一年前的暑假，我在学校闲得无聊，开始学习网络上炒得很火的python，大概几个下午的时间，简单阅读了一下它的语法，我觉得很好入手。可惜自此之后，便很少用了，也就生疏了。第二次邂逅，是在大三下学期的地球物理基础实验课上，我记得很清楚，萌萌的万柯松老师眯着眼睛给我们上了一堂python课，名字叫《用Python整合地球物理软件》，很惭愧，讲了什么我的确忘得一干二净了，但是仍然记得万老师满面春风地说出那句脍炙人口的话：人生苦短，我用Python！给我小小的心灵造成了很大的震撼。

仍然很惭愧，自此以后，我也没逮着机会好好用python。转折点是在2018年的夏天，我有幸前往闻名世界的加州理工学院地震实验室（请允许我时常提起这件事情，我把它作为一件很光荣的可以吹很久的事迹）。当时，我需要处理地震数据，需要接手博士生Ethan的代码，他酷爱使用python，于是，我真正开始了长时间使用python。之前我一般采用matlab进行绘图，效果很差，虽然matlab的计算能力强大，但不好意思，绘制出的图片，我也实在不是谦虚，非常差，以至于我组会作进展报告时，羞愤不已。

接手Ethan的代码后，我才知道原来python的matplotlib能画出这么好看的图，而且我还用它做了很多gif动图，可以说功能非常强大了。我如饥似渴地学习python的各种API。回国后不久，我开始了我的第一个开源项目，PySeislib的开发，到现在为止，已经完成了第一个项目，PyE-Click，跨平台多情景的划词翻译工具。目前我并没有急着开放源代码，项目还需要测试，更重要的原因是，我借由这个项目，找到了python新的学习方向——网络爬虫。

网络爬虫，现在来讲，已经过气了，因为能爬到的有价值的东西已经很少了。

为什么我要继续学习网络爬虫？答案是我真的很闲。。。。。。

言归正传，网络爬虫还是能做很多事情了，只要你敢想，我就能做。另外，学习网络爬虫，能系统地回顾之前我学过的web前段技术和数据库的相关知识。

1.学习思路

使用范传辉编著的《Python爬虫开发与项目实践》作为教材。这本书使用的是Python2.7，但现在主流的Python版本已经是3.6了，很多地方不兼容，我决心对其中内容进行修改，以飨读者。这本书的信息量之大，令我激动不已，用精简的笔法cover了web编程的方方面面，推荐各位一读。

1.1 开发环境

Anaconda + Python 3.6.5+Ubuntu 18.04/Red hat

我觉得我有一个优势就是我所在的实验室有一个闲置的大服务器，因为换了更好的新服务器，老服务器没人用了，这样爬虫可以放在服务器上24小时不间断地爬取并下载资源，当然我肯定不做违法违纪的事情，不然我老板知道了一定会开除我的。

1.2 预备知识

开发网络爬虫需要一些预备知识，现列举如下：

  • Python基本语法和相关库的使用
  • Web基本知识
    • Html
    • CSS
    • JS
    • Xpath
    • Json
    • Ajax
  • 数据库知识

说实话，上面很多东西我都是半桶水甚至不会，我也不可能一项一项去熟练，然后再去学习网络爬虫，我采取的是边学爬虫边补基础的办法，我的计算机底子太差了，毕竟不是科班出身，我需要去听一些计院的专业课，也许。

所以这份笔记的布局比较诡异，我把上面列举的诸多东西放在附录里面，并且加上一些无关爬虫的bonus，而附录也随时更新，正文直接从python网络爬虫开始讲起。主要结构如下：

PYSPY附录

  • 超文本标记语言Html和CSS
  • 动态Web脚本Javascript
  • Xpath与JSON
  • 浅谈万维网
  • 再学数据库

其他

  • Markdown笔记（bonus）
  • Latex笔记（bonus）

  • C++学习笔记（主要是用来熟悉OOP面向对象编程，bonus）

  • C语言笔记（bonus）

  • Fortran计算与IDL制图笔记（bonus）

  • Matlab笔记

  • Java学习笔记

  • Exel制图

所以，现就这样了，开始吧！
